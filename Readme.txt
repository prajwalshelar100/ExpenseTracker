Expense Tracker UI
Expense Tracker UI is a Java Swing application that allows users to manage their expenses by recording and categorizing them. The program provides a graphical user interface to add, view, summarize, and clear expenses.

Introduction
This Java application enables users to track their expenses conveniently through a simple graphical interface. It utilizes Swing components to facilitate adding new expenses, viewing all recorded expenses, summarizing expenses by category, and clearing recorded expenses.

Features
Add Expense: Users can input a description, amount, and category for a new expense.
View Expenses: Provides a comprehensive view of all recorded expenses categorized by their respective categories.
Expense Summary: Allows users to obtain the total expenses for a specific expense category.
Clear Expenses: Clears all recorded expenses.
Display of Recent Expenses: Shows the description of the last 5 recorded expenses in the UI.
Usage
To use the Expense Tracker UI:

Run the Java application.
Use the provided buttons in the UI to add, view, summarize, or clear expenses.
Input expense details such as description, amount, and category when prompted.
The UI will display the recent expenses and respective operations as requested.
Installation
Prerequisites
Java Development Kit (JDK) installed on your system
Steps
Clone or download the repository.
Compile the Java code using javac ExpenseTrackerUI.java.
Run the compiled code using java ExpenseTrackerUI.
